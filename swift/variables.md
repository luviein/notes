## var
```swift
var greeting = "Hello, playground"
```

creates a variable that can be changed over time
can only store string after its initialised 

eg:
```swift
name = "Rebecca"
name = "Keeley"
```

## let
```swift
let character = "Daphne"
```

let = constant, cannot change

## print
```swift
var playerName = "Roy"
print(playerName)
```
